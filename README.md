# ansible-relp

An Ansible role for installing RELP support for Rsyslog.

## Role Variables

- `relp_version` - RELP version
- `relp_host` - Host to send RELP messages (default: `localhost`)
- `relp_port` - Port to sent RELP message (default: `20514`)
